# Introduction

A small project made with the help of React-diagrams [here](https://github.com/projectstorm/react-diagrams).

A tool to make flows and diagrams.

![](./assets/demo.png)

### Features

- Drag and drop the whole frams
- Place the nodes anywhere
- Logs the changes

## How to run

```
npm install
npm start
```
